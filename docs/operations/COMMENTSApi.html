This is the API

добавление комментария к отзыву

Добавить новый комментарий для отзыва. Права доступа: **Аутентифицированные пользователи.**

URL

/api/v1//titles/{title_id}/reviews/{review_id}/comments/

HTTP Method

POST

Response Type

Comment

Parameters

  • Path: title_id Integer

    ID произведения

  • Path: review_id Integer

    ID отзыва

  • Body: body Comment(Comment)

получение комментария к отзыву

Получить комментарий для отзыва по id. Права доступа: **Доступно без токена.**

URL

/api/v1//titles/{title_id}/reviews/{review_id}/comments/{comment_id}/

HTTP Method

GET

Response Type

Comment

Parameters

  • Path: title_id Integer

    ID произведения

  • Path: review_id Integer

    ID отзыва

  • Path: comment_id Integer

    ID комментария

получение списка всех комментариев к отзыву

Получить список всех комментариев к отзыву по id Права доступа: **Доступно без токена.**

URL

/api/v1//titles/{title_id}/reviews/{review_id}/comments/

HTTP Method

GET

Response Type

inline_response_200_5

Parameters

  • Path: title_id Integer

    ID произведения

  • Path: review_id Integer

    ID отзыва
