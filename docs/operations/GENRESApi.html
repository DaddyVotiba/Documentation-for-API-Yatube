This is the API

добавление жанра

Добавить жанр. Права доступа: **Администратор**. Поле `slug` каждого жанра должно быть уникальным.

URL

/api/v1//genres/

HTTP Method

POST

Response Type

GenreRead

Parameters

  • Body: body Genre(Genre)

получение списка всех жанров

Получить список всех жанров. Права доступа: **Доступно без токена**

URL

/api/v1//genres/

HTTP Method

GET

Response Type

inline_response_200_2

Parameters

  • Query: search String

    Поиск по названию жанра

удаление жанра

Удалить жанр. Права доступа: **Администратор**.

URL

/api/v1//genres/{slug}/

HTTP Method

DELETE

Response Type

Parameters

  • Path: slug String

    Slug жанра
